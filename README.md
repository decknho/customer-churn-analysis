# Análise de Churn de Clientes

## Objetivo do Projeto
Este projeto tem como objetivo analisar o comportamento de churn dos clientes e identificar possíveis fatores relacionados ao cancelamento.

---

## Tecnologias Utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## Dataset
O conjunto de dados contém informações dos clientes, como:
- Gênero
- Tipo de contrato
- Tempo como cliente
- Método de pagamento
- Valor mensal
- Status de churn

---

## Análises Realizadas
- Limpeza de dados
- Tratamento de valores nulos
- Análise exploratória (EDA)
- Identificação de outliers
- Análise de churn
- Visualização de dados

---

## Principais Insights
- Clientes com contrato mensal apresentaram maior taxa de churn.
- Clientes novos tendem a cancelar mais.
- Clientes casados aparentam ter menor taxa de churn.
- Métodos de pagamento eletrônicos apresentaram maior ocorrência de cancelamento.
